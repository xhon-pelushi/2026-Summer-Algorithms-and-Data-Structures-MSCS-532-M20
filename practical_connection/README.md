# Practical Connection Assignment

A reflection connecting course concepts to my current role as IT Manager at
TWP Clothing:

- **Divide-and-conquer as a migration strategy** — decomposing the RLM to
  BlueCherry ERP migration by data domain and location, mirroring how a
  divide-and-conquer algorithm splits and recombines subproblems.
- **Hashing and collision handling** — keying records across RLM,
  BlueCherry, and Shopify without collisions, and sizing catalog/customer
  tables against the load-factor intuition from the hash tables unit.
- **Asymptotic reasoning for vendor and tool evaluation** — comparing
  catalog search and lookup performance at scale instead of on a small demo
  dataset, and choosing data structures deliberately in internal .NET tools.

No source code — this assignment is a written reflection only.
